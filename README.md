# AI Assignment 3: Dijkstra + UGV Path Planning

This repository contains solutions for three programming tasks:

1. Dijkstra shortest-path search on an India road network.
2. UGV navigation in a static-obstacle battlefield grid.
3. UGV navigation in a dynamic, partially-known obstacle environment.

---

## Project Structure

- `Dijkstra/dijkstra.py`  
  Dijkstra shortest path on an India-wide road graph (with optional CSV input for open-source road datasets).

- `UGV_Static_Obstacles/ugv_static.py`  
  Static-obstacle UGV planner on a `70x70` grid with 3 obstacle densities and Measures of Effectiveness (MOE).

- `UGV_Dynamic_Obstacles/ugv_dynamic.py`  
  Dynamic-obstacle UGV simulator with local sensing and online replanning.

- `UGV_Static_Obstacles/static_moe_results.csv`  
  Generated MOE output for static experiments.

- `UGV_Dynamic_Obstacles/dynamic_moe_results.csv`  
  Generated MOE output for dynamic experiments.
